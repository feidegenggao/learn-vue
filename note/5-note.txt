2.css设定的优先级？
	CSS的设定的优先级，是有详细的设定就用详细的设定。
	css的设置有这种：class = "a b c"如果有定义
	.a.b.c {
		各种css设定
	}
	这是最匹配的，其中.a.b.c的顺序没有影响
	.a .b {
		这种的意思是.a嵌套的.b的布局
	}
